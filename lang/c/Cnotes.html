goto语句 ​

goto最好不要使用，会严重扰乱程序的流程

使用goto的时候，我们必须要在goto语句之前，定义一个标号

然后在goto语句中，指定标号，这样程序才能找到标号的位置，然后跳转到标号的位置

结合标号，可以跳转到代码中的任意地方，一般只用于嵌套循环的跳出

下面是一个简单的goto语句使用:

c
int main()
{

  for (int i = 0; i < 10; i++)
  {
    for (int j = 0; j < 10; j++)
    {
      printf("%d %d\n",i,j);
      goto a;
    }
  }
  //标号
  a: printf("a,goto执行完毕,循环结束");
}

  • 第一个参数位置填入的是最终的表达效果，也就是要输出的东西

下方未标注的数据类型所占内存都是以windows操作系统计算的 macos 和 linux 不一定是一样的

作用:

  1. 用来定义变量中能存储什么类型的数据

整形 ​

不同的数据类型所表示范围和内存大小都不一样，由编译器来决定的，可以用sizeof来确定

会遵守一个原则: short<=int<=int<=long<=long long

c语言中，默认整数类型为int

任何定义了数据类型的变量超出取值范围都会报错

整数取值范围(32/64位)内存(字节)
short-32768~327672
int-2147483648~2147483647(10位数)4
long(32位)-2147483648~2147483647(10位数)4
long(64位)-9223372036854775808~9223372036854775807(19位数)8
long long(c99标准)-9223372036854775808~9223372036854775807(19位数)8

==取余(%)细节==

  1. 运算的数据必须全部都是整数
  2. 获取余数的政府，是跟第一个数字保持一致的
c
int main()
{
  //运算的数据必须全部都是整数
  printf("%d",10%3);;
  //获取余数的政府，是跟第一个数字保持一致的
  printf("%d",10%3); // 1
  printf("%d",-10%3); // -1
  printf("%d",10%-3); // 1

  return 0;
}

  1. 取值范围小的，和取值范围大的计算，小的会自动提升为大的，再进行计算

==case穿透==

穿透规则:

  1. 根据小括号中的表达式的结果去匹配对应的case
  2. 执行对应case里面的代码
  3. 如果在执行的过程中，遇到了break，那么直接结束整个switch 但是如果没有遇到break，就会继续执行下面case中的代码 直到遇到break，或者将整个switch中的所有代码全部执行完了，才会结束
  4. case穿透只会往下穿透，不会往上穿透

case穿透的作用案例:
