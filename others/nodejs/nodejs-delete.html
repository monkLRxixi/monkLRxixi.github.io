Nodejs卸载 ​

要彻底删除 Node.js，你需要删除几个关键的文件和文件夹。以下是删除 Node.js 的一般步骤：

在 Windows 上删除 Node.js ​

卸载 Node.js ​

  • 通过 "控制面板" -> "程序和功能" 打开程序列表。

  • 在列表中找到 Node.js，右键点击选择 "卸载"。

删除安装目录 ​

  • 手动删除 Node.js 的安装目录。通常情况下，Node.js 默认安装在 C:\Program Files\nodejs 或 C:\Program Files (x86)\nodejs。

删除环境变量 ​

  • 可以通过以下步骤手动删除环境变量中与 Node.js 相关的条目：
  • 在 Windows 搜索栏中搜索 "环境变量" 并打开 "编辑系统环境变量"。
  • 在弹出的窗口中，点击 "环境变量" 按钮。
  • 在 "系统变量" 列表中找到名为 Path 的变量，编辑并删除其中与 Node.js 相关的路径。
  • 检查 "系统变量" 和 "用户变量"，删除任何与 Node.js 相关的变量。

删除其他相关文件和文件夹 ​

  • 检查可能存放 npm 全局安装包的文件夹。默认情况下，全局包安装在

C:\Users\YourUsername\AppData\Roaming\npm 或 C:\Users\YourUsername\AppData\Roaming\npm-cache。

在 macOS 上删除 Node.js ​

使用终端卸载 Node.js ​

  • 打开终端应用。
  • 运行以下命令以卸载 Node.js 和 npm：
bash
sudo rm -rf /usr/local/{lib/node{,/.npm,_modules},bin,share/man}/{npm*,node*,man1/node*}
sudo rm -rf ~/{.npm,.npmrc,.node*,.npm*}
sudo rm -rf /opt/local/bin/node /opt/local/include/node /opt/local/lib/node_modules/
sudo rm -rf /usr/local/{lib/node{,/.npm,_modules},bin,share/man}/{npm*,node*,man1/node*}
sudo rm -rf ~/{.npm,.npmrc,.node*,.npm*}
sudo rm -rf /opt/local/bin/node /opt/local/include/node /opt/local/lib/node_modules/

删除安装目录 ​

  • 手动删除 Node.js 的安装目录。通常情况下，Node.js 默认安装在 /usr/local/bin/node 或者 /usr/local/bin/nodejs。

删除环境变量 ​

  • 编辑 ~/.bash_profile 或 ~/.bashrc 文件，删除 Node.js 相关的路径。使用 sudo nano ~/.bash_profile 或 sudo nano ~/.bashrc 命令来编辑文件。

删除其他相关文件和文件夹 ​

  • 检查可能存放 npm 全局安装包的文件夹。默认情况下，全局包安装在 ~/.npm 或~/.npm-global。

记得在进行上述步骤时要小心谨慎，确保删除正确的文件和文件夹。彻底删除 Node.js 后，你也可以重新安装 Node.js 的其他版本或者重新配置环境。
